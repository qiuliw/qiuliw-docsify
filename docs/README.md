## 关于我

萨默赛特・毛姆写道：“任何一把剃刀都自有其哲学。” 大约是说，无论何等微不足道的举动，只要日日坚持，从中总会生长出独属于自己的观念与认知。我十分认同这段话。

我是一名普通后端开发者，业余偏爱钻研 P2P 架构与去中心化体系，认可密码朋克以加密捍卫数字自主的理念，反感中心化平台对数据与身份的垄断和审查。

我属于不把思绪写成文字就无法理清思路的人。没有高深的理论积累，也谈不上成熟的实践成果，只是日复一日沉在反复试错。这个博客便是我的 “长跑”，一字一句记下调试、研读、思辨的碎片，在持续打磨中沉淀独属于自己的浅薄认知。

希望于后人有点益，弥补过往从他人博客的CV，只进不出，传承CV的分享精神。

言语乏力，也借此锻炼，望观者海涵。
